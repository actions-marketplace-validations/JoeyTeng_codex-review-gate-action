# Support

Use GitHub issues in this repository for:

- action bugs
- incorrect `codex/review-gate` status behaviour
- documentation gaps
- Marketplace listing issues
- compatibility problems with target repository workflows

If a pull request receives problematic AI-generated Codex review content, use GitHub's normal feedback or reporting tools for that specific comment or review. Open an issue here only when the problem affects this action's marker, status, retry, or signal-handling behaviour.
